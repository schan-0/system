## 대소문자 변환 프로그램

이 프로그램은 사용자가 문자를 입력하면 해당 문자가 대문자인지 소문자인지에 따라 변환된 값을 출력합니다. `0`을 입력하면 프로그램이 종료됩니다.

---

### 🚀 기능

* **대소문자 변환:** 입력된 문자가 대문자이면 소문자로, 소문자이면 대문자로 변환하여 출력합니다.
* **종료 기능:** `0`을 입력하면 프로그램을 종료합니다.
* **유효성 검사:** 알파벳이 아닌 다른 문자가 입력될 경우 "유효한 입력이 아닙니다."라는 메시지를 출력합니다.

---

### 🛠️ 빌드 및 실행 방법

이 프로그램을 빌드하고 실행하려면 C 컴파일러(예: GCC)가 필요합니다.

1.  **소스 코드 저장:** 위의 코드를 `case_converter.c` 파일로 저장합니다.

2.  **컴파일:** 터미널 또는 명령 프롬프트에서 다음 명령어를 사용하여 컴파일합니다.

    ```bash
    gcc case_converter.c -o case_converter
    ```

3.  **실행:** 컴파일이 완료되면 다음 명령어를 사용하여 프로그램을 실행합니다.

    ```bash
    ./case_converter
    ```

---

### 💻 사용 예시

프로그램을 실행하면 `문자 입력 :` 메시지가 반복해서 나타납니다.

```
문자 입력 : A
A의 소문자는 a입니다.
문자 입력 : b
b의 대문자는 B입니다.
문자 입력 : 5
유효한 입력이 아닙니다.
문자 입력 : 0
```

### 💡 코드 설명

* `main()`: 프로그램의 진입점입니다.
    * `while(1)`: 무한 루프를 통해 사용자가 `0`을 입력하기 전까지 계속 문자를 입력받습니다.
    * `getchar()`: 사용자로부터 한 문자를 입력받습니다.
    * `while ((tmp = getchar()) != '\n');`: 입력 버퍼에 남아있는 개행 문자(`\n`)를 비워주는 역할을 합니다. 이는 `getchar()`를 사용할 때 발생하는 흔한 문제를 방지합니다.
    * `if (input == '0') return 1;`: 입력이 `0`이면 프로그램을 종료합니다.
    * `isupper(input)`: 입력된 문자가 대문자인지 확인합니다.
    * `islower(input)`: 입력된 문자가 소문자인지 확인합니다.
    * `tolower(input)`: 대문자를 소문자로 변환합니다.
    * `toupper(input)`: 소문자를 대문자로 변환합니다.
    * `else`: 알파벳이 아닌 경우 "유효한 입력이 아닙니다."를 출력합니다.
